行测(2)——推理题

王致远

2018/10/23

1. 图形推理题

注意：

  • 九宫格问题一般横着找规律；
  • 有些存在交替规律或递增递减规律；
  • 九宫格问题：第一与第二存在相同规律，第二与第三存在相同规律

规律总结：

1.1 图形元素数量规律

规律：相同，递增，递减

图形：

  • 特殊形状
  • 直线
  • 圆弧
  • 封闭区间

一笔画出还是两笔画出；当排除了其他可能性时，考虑此可能性

1.2 图形位置规律

旋转，翻转，平移

图形：

  • 直线与曲线位置；
  • 直线与圆交点个数；
  • 点与线的位置关系；
  • 图形元素之间包含还是分离；

旋转： - 对称轴依次旋转一定角度； - 点绕九宫格旋转； - 线端点绕内九宫格旋转；

1.3 图形性质规律

  • 形状：直角，锐角，钝角三角形
  • 对称性：
    • 轴对称
    • 中心对称
  • 面积

1.4 图形运算规律

  • 叠加
  • 相减
  • 去异存同
  • 去同存异

1.5 图形元素轮换规律

当看到一些元素重复出现时，应考虑是否存在轮换规律

1.6 图形折叠和拆分规律

弄清某两个相邻面或对立面的特征，相对的两个面在立体图中只能出现一个。

2. 定义判断

2.1 关键信息类

将定义中的关键词找到，作为判断的依据。

2.2 代入定义类

将选项直接带入定义中进行判断。

3. 类比推理题

  1. 注意正反义词顺序；
  2. 注意递进规律还是递减规律；
  3. 注意词义的褒义贬义；
  4. 将名词用动词连接，选择相同动词的；

4. 逻辑判断题

4.1 最能反驳上述结论

  1. 论据不充分；
  2. 论据是错的；
  3. 提出另外一种原因解释被认为是最强质疑；
  4. 其他因素影响

如果提出新的可能性，但没有所支持的证据，或证据链不充分，则反驳力度较弱。

4.2 最能支持上述观点

  1. 原因解释比时间重合更能说明观点（从文科生的观点）；
  2. 新的实验和证据；
  3. 新的发现；
  4. 补充关键信息；

4.3 要得到上述结论需补充的前提

  1. 在论据和论点之间建立联系；
  2. 类比不一定成立；

4.4 最不能支持

看选项中哪些是支持的，用排除法。最不能支持的可能不是反驳项，而是无关项。

4.4 任务分配问题

列表求解，行为可能性，列为任务，每一行代表一种分配方案。
